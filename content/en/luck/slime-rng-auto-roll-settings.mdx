## Troubleshooting Common Auto Roll Issues

Even with the best **slime rng auto roll settings**, you may encounter bugs. The most common issue in 2026 is the "Roll Freeze," where the UI stops updating despite the script running in the background. This is usually caused by a memory leak or a conflict between the game's native auto-roll and a third-party script's auto-roll.

To fix this, disable the native auto-roll button before launching your script. This ensures that only one command source is sending requests to the server. Additionally, make sure your "Walk Speed" isn't set so high that it triggers the game's anti-cheat, which can disconnect you and end your farming session.
