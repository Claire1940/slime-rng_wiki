Players often bundle all failures into **slime rng expired codes**, but different errors need different fixes.

### Advanced troubleshooting routine (2-minute test)

- Rejoin a fresh public server.

- Try one redemption attempt.
- If still invalid, wait 30 minutes and test once more.
- If repeated “expired” appears, move it to archive.

This workflow gives you cleaner results than testing the same code 10+ times in one laggy session.
